Simple Blender Addon (for Blender 5.1, could work on other versions aswell, idk) to Import/Export Skeletal Animations in Blender using Nintendo's bfsha format.
(Via the Import/Export context menu)

Works with Splatoon 3 Animations, i haven't tested other game's anims, but it should work aswell?

There might be a rotation issue where bones aren't properly rotated when exporting (90° rotated in a false direction), you can fix that by just rotating the Root Bone in Blender before exporting.
